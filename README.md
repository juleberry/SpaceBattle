# SpaceBattle
#### Text-based game (played in the Developer Tools Console)
#### ** DESKTOP ONLY **
This game was created as part of my cohort in Per Scholas' software engineering program.

### "Greetings!
### Earth has been attacked by a horde of aliens!
### You are the captain of the USS CRYSTAL on a mission to destroy every last alien ship."

### Gameplay:
For best experience, please use Chrome browser.
Open Chrome "Menu" > "More Tools" > "Developer tools"
Then, select the "Console" tab.

Shortcut: CTRL + SHIFT + "I" (to open "Developer tools") then select the "Console" tab.
